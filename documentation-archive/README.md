## Why Archive?

These documents were moved to the archive to:

1. **Reduce clutter** - Keep root directory focused on current documentation
2. **Preserve history** - Maintain record of project evolution
3. **Prevent confusion** - Avoid outdated information being used as current reference
4. **Maintain context** - Keep historical decisions and rationale accessible

## Usage

Reference these documents when:

- Understanding historical decisions
- Reviewing project evolution
- Researching past implementations
- Auditing previous compliance states

**Note:** Do not use these documents as current reference. Always refer to the main documentation for current standards and practices.
